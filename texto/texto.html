Rede de computadores

Uma rede de computadores é um conjunto de equipamentos inteligados se maneira a trocarem informações e compartilharem recursos, como arquicos de dados gravados, impressoras, modems, software e outros

Importancia das redes de computadosres

Proporcionam economia, compartilhamento de recursos, como impressoras de projetores, bem como de serviços, que é o caso da internet. Permite tambem, por meio do compartilhamento de arquivos, o trabalho em equipe dos colaboradores.

LAN WAN

LAN (Local Area Network) é uma rede de computadores que cobre uma area limitada, geralmente um edificio ou um campus. É usado para conectar dispositivos em um espaço fisico limitado, como uma casa, escritorio ou sala de aula. Já (Wide Area Network) é uma rede de computadores que abrange uma area geografica maior, geralmente uma cidade, um estado ou até mesmo um pais. Ela é usada para conctar dispositivos que estao fisicamente distantes entre si.
